Application Settings
Default Pusheen Settings

Here you can set the pusheens that will be shown by the default widget window. imgUrl links to the image that you want to be shown, while text will be what pusheen says to you.

Setting the text value empty will result in only showing the gif image you set in the imgUrl property.

Please remember that sticker gifs work the best for this widget as they offer a trasparent background. You can find some nice ones here.
